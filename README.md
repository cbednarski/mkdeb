# mkdeb

A Go CLI and library for building debian packages.

To use the CLI:

    go get github.com/cbednarski/mkdeb
    mkdeb

To use as a library:

    go get github.com/cbednarski/mkdeb/deb
    godoc -http=:8080

Browse to <http://localhost:8080/pkg/github.com/cbednarski/mkdeb/deb/>
